#Pokedux

A simple pokedex build using React and Redux.  
Uses http://pokeapi.com as the api.

http://pokedux.jamesmcsweeny.com

##Local Development

1. `npm install`
2. `npm start`  
3. Your site will run at http://localhost:8080

##Production Build

1. `npm run build`
2. The bundled files will be in the /dist folder
